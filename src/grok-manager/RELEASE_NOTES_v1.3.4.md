# Grok Manager v1.3.4

## 变更

- **处理筛选**：对当前状态筛选一键测活 / 删除 / 解禁（适合大批 403）
- **auth.list 缓存** 约 8 秒，减轻凭证列表/全选时的主机压力；删除后自动失效
- **进度条 ETA**：大批量测活/删除显示剩余时间估算，轮询更顺
- 批量任务运行时暂停隔离 15s 自动刷新，避免抢请求

## 使用提示

1. 隔离页点 **403** 卡片筛选  
2. 点 **处理筛选** → 确认测活或删除  
3. 看底部进度条（含 ETA）
