# Grok Manager v1.3.3

## 变更

### 体验收敛
- 测活页：**凭证 | 结果** 两 Tab（去掉重复的「全量结果 / 测活结果 / 凭证列表」三套）
- 结果页用**会话**切换：全量测活 / 勾选测活历史
- 隔离页精简主操作，低频操作收进「更多」
- 隔离支持**全选筛选结果**（跨页）

### 大批量
- 测活（>5 条）后台异步 + **底部进度条**（done/total）
- 删除（>15 条）同样异步 + 进度
- 完成后自动刷新并跳到结果会话

### 其它
- 凭证列表勾选测活、测活结果统计
- 纯 SSO / email----sso 入库（既有）

## 下载

- Linux amd64/arm64 `.so`
- Windows amd64 `.dll`
